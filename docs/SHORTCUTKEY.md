## デスクトップ・ウィンドウ操作に関するショートカットキー一覧

### ウィンドウの配置・サイズ変更

| ショートカットキー              | 動作                                                           | 由来・覚え方 |
| :------------------------------ | :------------------------------------------------------------- | :----------- |
| **`Win` + `↑`**                 | アクティブなウィンドウを最大化する                             |              |
| **`Win` + `↓`**                 | ウィンドウを元のサイズに戻す（最大化時）、または最小化する     |              |
| **`Win` + `←` / `→`**           | ウィンドウを画面の左半分・右半分にスナップ（配置）する         |              |
| **`Win` + `Alt` + `←` / `→`**   | ウィンドウを画面の1/3にスナップする                            |              |
| **`Win` + `Shift` + `←` / `→`** | アクティブなウィンドウを、隣のモニターへ移動する               |              |
| **`Win` + `Shift` + `↑`**       | ウィンドウの幅を維持したまま、画面の上下いっぱいに拡大する     |              |
| **`Win` + `Shift` + `↓`**       | 上下に拡大されたウィンドウを元のサイズに戻す、または最小化する |              |
| **`Win` + `Z`**                 | スナップレイアウトを開く（※Windows 11のみ）                    | Zone の Z    |

### デスクトップの表示・最小化

| ショートカットキー        | 動作                                                           | 由来・覚え方  |
| :------------------------ | :------------------------------------------------------------- | :------------ |
| **`Win` + `D`**           | デスクトップを表示・非表示にする                               | Desktop の D  |
| **`Win` + `M`**           | すべてのウィンドウを最小化する                                 | Minimize の M |
| **`Win` + `Shift` + `M`** | 最小化されたすべてのウィンドウを元に戻す                       |               |
| **`Win` + `Home`**        | アクティブなウィンドウ以外の「すべてのウィンドウ」を最小化する |               |
| **`Win` + `,`**           | キーを押している間だけ、デスクトップを一時的に表示する         |               |

### ウィンドウの切り替え・タスクバー操作

| ショートカットキー                   | 動作                                                                                                                | 由来・覚え方             |
| :----------------------------------- | :------------------------------------------------------------------------------------------------------------------ | :----------------------- |
| **`Alt` + `F4`**                     | アクティブなウィンドウ（アプリ全体）を閉じる。デスクトップ表示時に押すと「Windowsのシャットダウン」ダイアログを開く |                          |
| **`Ctrl` + `F4`** / **`Ctrl` + `W`** | アプリ全体ではなく、現在開いている「タブ」や「ドキュメント」のみを閉じる                                            | **W**indow / **W**rap up |
| **`Alt` + `Space`**                  | アクティブなウィンドウのコントロールメニュー（移動、サイズ変更、最大・最小化など）を左上に開く                      | **Space** menu           |
| **`Alt` + `Space`** → **`X`**        | （コントロールメニュー経由で）アクティブなウィンドウを最大化する                                                    | Ma**x**imize             |
| **`Alt` + `Space`** → **`N`**        | （コントロールメニュー経由で）アクティブなウィンドウを最小化する                                                    | Mi**n**imize             |
| **`Alt` + `Tab`**                    | 開いているアプリ（ウィンドウ）の一覧を表示して素早く切り替える                                                      |                          |
| **`Ctrl` + `Alt` + `Tab`**           | アプリの切り替え画面を固定表示する（矢印キーで選択し `Enter` で決定）                                               |                          |
| **`Win` + `Tab`**                    | タスクビューを開く（起動中の全ウィンドウと仮想デスクトップを一覧表示）                                              |                          |
| **`Alt` + `Esc`**                    | ウィンドウを開いた順番で直接背面に送る・前面に呼び出す（一覧画面なし）                                              | Escape の ESC            |
| **`Win` + `T`**                      | タスクバー上のアプリを順番に選択する（`Enter`キーで開く）                                                           | Taskbar の T             |
| **`Win` + `数字 (1~9)`**             | タスクバーに左からピン留めされているアプリを起動、またはアクティブにする                                            |                          |

### 仮想デスクトップ操作

| ショートカットキー             | 動作                                       | 由来・覚え方                 |
| :----------------------------- | :----------------------------------------- | :--------------------------- |
| **`Win` + `Ctrl` + `D`**       | 新しい仮想デスクトップを作成する           | Desktop の D                 |
| **`Win` + `Ctrl` + `←` / `→`** | 作成した仮想デスクトップを左右に切り替える |                              |
| **`Win` + `Ctrl` + `F4`**      | 現在開いている仮想デスクトップを閉じる     | ウィンドウを閉じる(F4)と同じ |

### 本ツールの追加機能

上記で空いているショートカットキーを活用して、以下の機能を割り当てます。

| ショートカットキー                        | 動作                                                               | 由来・覚え方                 |
| :---------------------------------------- | :----------------------------------------------------------------- | :--------------------------- |
| **`Win` + `Ctrl` + `Alt` + `←` / `→`**    | 全てのウィンドウの比率を維持して、他のモニタと入れ替える           |                              |
| **`Win` + `Ctrl` + `Alt` + `数字 (2~9)`** | プライマリモニタと入力したキー番号のモニタのウィンドウを入れ替える |                              |
| **`Win` + `Ctrl` + `Alt` + `Z`**          | アンドゥ（ウィンドウの入れ替え操作を元に戻す）                     | 「元に戻す」で一般的なキー Z |
| **`Win` + `Ctrl` + `Alt` + `P`**          | 【v0.2】アクティブなウィンドウを現在のモニターにピン留めする       | Pin の P                     |
| **`Win` + `Ctrl` + `Alt` + `E`**          | 【v0.2】ターゲットモニターの全ウィンドウを他モニターへ一斉退避する | Escape の E                  |
| **`Win` + `Ctrl` + `Alt` + `G`**          | 【v0.2】全ウィンドウをターゲットモニターへ一極集中（集約）する     | Gather の G                  |
| **`Win` + `Ctrl` + `Alt` + `A`**          | 【v0.2】ウィンドウのスマート整列メニューを表示する                 | Align / Arrange の A         |
